請閱讀 docs/examples/webssh-external-agent-skill/SKILL.md，增加 `webssh-external-agent` local skill；若使用者要求操作 WebSSH，從 WebSSH startup banner 或該 skill 取得 active Python、`scripts/webssh_agent_cli.py`、`scripts/webssh_agent_jsonl.py`、`scripts/webssh_agent_repl.py`、`webssh_external_agent_handoff.json` 的絕對路徑，不要猜 port、URL 或 token。
使用時先確認 handoff 是瀏覽器 Agent UI 新 mint 的 local secret，不印出 token；先跑 `hello` 並只依 typed JSON fields 決策，連續機器操作優先用 JSONL client，單步優先 `send-wait`/`screen --tail-lines`/`screen --region`，ANSI redraw 太吵才加 `tail --strip-ansi` 或 `send-wait --strip-ansi`；若 token unauthorized/expired/revoked 或 URL 不符，請使用者重新 mint token。
